El derecho busca establecer una proporción entre las acciones y reacciones de forma tal que no escale la situación. Cohersitivamente impuesta por el estado.

Definición Formal de Cátedra:

>El Derecho comprende el conjunto de normas de conducta impuestas por el
Estado con carácter obligatorio y conforme a criterios de justicia. Define
límites, atribuye poderes y garantiza soluciones pacíficas a los conflictos
sociales.
